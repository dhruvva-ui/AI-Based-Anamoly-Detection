The Intelligent Healthcare Anomaly Detection System is a real-time monitoring and decision support platform developed to identify unusual patterns in patient vital signs and provide early health risk alerts. Unlike conventional healthcare monitoring solutions that depend on fixed threshold values or manual review after an incident, this system continuously processes patient health data and applies machine learning techniques to detect irregularities as they occur. When an anomaly is detected, the system records the event in a PostgreSQL database and displays the information on an interactive dashboard built using Flask. In critical situations, automated email notifications are sent to healthcare staff to ensure quick response and timely intervention.This project highlights the integration of multiple technologies including machine learning models, real-time data processing, backend API development, and data visualization dashboards to address practical challenges in healthcare monitoring.
